Audiogen 2 Movie Player X - |guideX|
 
An easy to use and powerfull media playback control. Easily incorperate Playback for *.m3u, *.m4a, *.avi, *.mpg, *.mpeg, *.mpe, *.mp3, *.mp2, *.mp1, *.wav, *.aif, *.aiff, *.aifc, *.au, *.mv1, *.mov, *.mpa, *.qt, *.snd, *.mpm, *.mpv, *.enc, *.mid, *.rmi, *.vob, *.wma, *.wmv into your programs written in Visual Basic, Delphi and generally any other development environment which will accept activex files.
 
Events:
AGDoubleClick() ' Triggers when user double clicks the movie window
AGRightClick() ' Triggers when user right clicks on movie window
AGMovieOpened(lFileName as String) ' Triggers when a media file is opened. lFilename was the file that was opened
AGMovieClosed() ' Triggered when the movie is closed
AGMovieStopped() ' Triggered when the movie is stopped
AGMoviePaused() ' Triggered when the movie is paused
AGMovieResumed() ' Triggered when the movie is unpaused
AGMoviePlay() ' Triggered when the movie is played
 
Methods:
ChangeMoviePosition(lSecond as Long) ' Changes movie position to the second specified in lSecond
ChangePlayRate(lValue as Long) ' Changes the speed which the movie is played back at.
CloseDoor() ' Closes the CD Door
CloseMovie() ' Closes the currently open movie
ForwardFrames(lNumFrames As Long) ' Goes forward the specified number of frames
ForwardSeconds(lNumSeconds as Long) ' Goes forward the specified number of seconds
FullScreen() ' Sets full screen on
GetFramesPerSecond() ' Returns rate of frames per second
GetStatus() ' Returns the status of the player
  ' 0 = Idle
  ' 1 = Playing
  ' 2 = Open
  ' 3 = Paused
  ' 4 = Stopped
  ' 5 = Error
GetVolume() ' Returns the current volume. This volume is not either the 'Wave' nor 'Master' volume in windows, rather is is the volume of the currently playing file
Mute(lToggle As Boolean) ' Sets the audio on or off
OpenCDDoor() ' Opens the CD rom door of the current cd drive
OpenMovie(lFileName As String) ' Opens a media file, or movie
PauseMovie() ' Pauses the currently playing movie
PlayMovie() ' Plays the current movie/media file
ResumeMovie() ' Resumes the current movie from a paused state
ReturnCurrentPosition() ' Returns the possition of playback in seconds
ReturnMovieFrames() ' Returns the number of frames in the currently open movie
ReturnTotalSeconds() ' Returns the total seconds of the currently open movie
RewindFrames(lNumFrames As Long) ' Rewinds the currently playing movie the specified number of frames
RewindSeconds(lNumSeconds As Long) ' Rewinds the currently playing movie the specified number of seconds
SetBlackDrop(lTopMargin As Integer, lSideMargin As Integer, lSideMargin2 As Integer, lBottomMargin As Integer) 'Sets a blackdrop for the image (a border on each side)
SetMovieWindowByhWnd(lhWnd As Long) ' Sets the object for the image to be played on
SetSize(lLeft As Long, lTop As Long, lWidth As Long, lHeight As Long) Sets the size of the movie image, best used in resize events
SetVolume(lVolume As Long) ' Sets the volume to the specified value
StopMovie() ' Stops the currently playing movie
